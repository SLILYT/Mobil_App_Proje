# Mobil_App_Proje
**MiniKatalogApp**, Flutter kullanılarak geliştirilmiş bir mobil uygulama projesidir. Uygulama, kullanıcıların ürünleri görüntülemesine ve temel katalog işlemlerini gerçekleştirmesine olanak sağlar.

Flutter 3.41.4 ile oluşturulmuştur.

# Özellikler
- Flutter ile platformlar arası (iOS & Android) uyumlu.
- Ürün listeleme ve detay görüntüleme.
- Temiz ve modern kullanıcı arayüzü.
- Kolayca genişletilebilir ve özelleştirilebilir yapı.
